FAQ

Warum wird manchmal kein Foto
angezeigt?

Der Mars Rover verfügt über mehrere Kameras und die API stellt immer nur für den jeweiligen Tag nur eine gewisse Anzahl an Fotos von den unterschiedlichen Kameras zur Verfügung. Dabei kommt es vor das es von manchen Kameras eine lange Zeit keine Fotos gibt. Je Älter das Datum, kann es auch sein das es ein anderer Rover war, diese hatten weniger Kameras oder auch andere.

Warum wird nichts mehr
angezeigt?

Die API Anfragen sind begrenzt, wird das Limit ausgereizt kommt es vor dass nichts mehr angezeigt wird. Sollte das Limit erreicht sein, warten Sie bitte eine Stunde, dann wird der Zähler zurückgesetzt und die API gibt wieder eine Antwort zurück. Es kann aber auch sein das einfach keine Bilder für diesen Tag existieren oder noch nicht zu Erde angekommen sind. Bei der InSight API gibt es nur Daten von den letzten paar Tagen.

Wie heißen die Abkürzungen der
Kameras?

1. Front Hazard Avoidance Camera (FHAZ)
2. Rear Hazard Avoidance Camera (RHAZ)
3. Mast Camera (MAST)
4. Chemistry and Camera Complex
   (CHEMCAM)
5. Mars Hand Lens Imager (MAHLI)
6. Mars Descent Imager (MARDI)
7. Navigation Camera (NAVCAM)
8. Panoramic Camera (PANCAM)
9. Miniature Thermal Emission Spectrometer
   (MINITES)

Warum das Wetter und den Mars Rover in Kombination?

Um jeden Tag neue Informationen zum Mars zu erhalten, bietet gerade der Mars Rover ein perfekte Lösung dafür, durch ihn ist es möglich neue Bilder vom Mars sehen zu können, z.B. Umgebung oder auch Probeanalysen von Gestein und Böden. Das Wetter gibt des Weiteren  ebenfalls täglich neue Informationen. Da nicht jeden Tag eine Bahnbrechende Entdeckung gemacht wird (auf dem Mars), soll diese App Ihnen eine Möglichkeit geben auf dem laufendem zu bleiben und eventuell selber eine neue Entdeckung für sich selbst zu machen oder auch bei neuen Entdeckungen, erfahren Sie es gleich mit als Erster! ;)
